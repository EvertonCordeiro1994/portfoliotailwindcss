Site Porsche 911

Este site foi desenvolvido com inspiração no icônico Porsche 911, capturando a elegância e o desempenho do carro em uma experiência web única. Criado durante uma aula prática, o projeto utiliza HTML, CSS e jQuery para entregar um design moderno e dinâmico, refletindo a sofisticação e o estilo do Porsche 911.

Site Dom Henrico Pizzaria

Este site é uma landing page desenvolvida durante um curso para a pizzaria fictícia Dom Henrico. Utilizando HTML, CSS e jQuery, o projeto apresenta um design moderno e inclui um formulário, onde os usuários podem preencher para ganhar um desconto exclusivo.
